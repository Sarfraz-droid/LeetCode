```
50. Pow(x, n)MediumImplement pow(x, n), which calculates x raised to the power n (i.e., xn).

&nbsp;
Example 1:

Input: x = 2.00000, n = 10
Output: 1024.00000


Example 2:

Input: x = 2.10000, n = 3
Output: 9.26100


Example 3:

Input: x = 2.00000, n = -2
Output: 0.25000
Explanation: 2-2 = 1/22 = 1/4 = 0.25


&nbsp;
Constraints:


	-100.0 &lt; x &lt; 100.0
	-231 &lt;= n &lt;= 231-1
	-104 &lt;= xn &lt;= 104

 
```

##Code
 ```
  class Solution {
public:
    double myPow(double x, int num) {
        double ans = 1;
        bool flag = false;
        long long n = num;
        if(n<0){
            flag = true;
            n = -n;
        }
        
        while(n){
            
            if(n&1){
                ans*=x;
            }
            
            x*=x;
            n=n/2;
        }
        
        return flag ?  1/ans : ans;
    }
}; 
```
